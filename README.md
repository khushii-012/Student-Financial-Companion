# Student Financial Companion 💰

An AI-powered financial management system designed for students living away from home.

## Features

- Budget Planning
- Expense Tracking
- Budget Alerts
- Expense Analytics
- Financial Health Score
- Month-End Spending Prediction

## Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Plotly
